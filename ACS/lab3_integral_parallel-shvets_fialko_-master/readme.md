## Prerequisites

### C++ prerequisites:
#### - cmake
#### - g++
#### - python3
### Compilation

```
$ ./compile.sh
```

### Run Python code

Python script run (where n - number of executions):
```
$ python3 ./prog_runner.py  n
```

### Run C++ code
Where 
* N - number of function.
* M - number of threads.
```
$ ./bin/integrate_parallel N ./funcN.cfg M
```
